Datenbanken sichern und wiederherstellen

Für gewöhnlich sind die Informationen, die in Passwortdatenbanken enthalten sind, sowohl sensibel als auch wichtig. Password Safe stellt einige Mechanismen zur Verfügung, um Passwörter zu sichern und wiederherzustellen. Diese Kapitel beschreibt Wege, um händisch Sicherungen ihrer Passwortdatenbanken zu machen. Password Safe kann auch so eingestellt werden dass automatisch Sicherungen gemacht werden, wenn die Datenbank gespeichert wird. Solche Sicherungskopien werden Zwischen-Sicherung genannt, und werden hier beschrieben.

Sicherung erstellen

Erstellen Sie eine Kopie der aktuellen Passwortdatenbank.

  1. Klicken Sie auf Sicherung erstellen
  2. Wählen Sie eine Speichestelle für die Sicherung
  3. Klicken Sie auf Speichern

Hinweis: Es wird dringend empfohlen regelmäßig Sicherungen der Passwortdatenbanken zu machen. Dies schützt Passwortlisten gegen Systemzusammenbrüche, Stromunterbrechungen und kaputte Dateien. Die Sicherung und die Datenbank sind mit demselben Master-Passwort verschlüsselt.

Wiederherstellen aus Sicherung

Die Wiederherstellung einer Passwortdatenbank aus einer Sicherung.

  1. Klicken Sie auf Wiederherstellung aus Sicherung
  2. Gehen Sie zum Verzeichnis wo die Sicherung sich befindet
  3. Klicken Sie auf Öffnen
    Ein Aufforderung zur Eingabe des Passwortes wird angezeigt.
  4. Geben Sie das Passwort ein und klicken Sie auf OK
Hinweis:
  • Die wiederhergestellte Datenbank ist mit der Password Safe Standarderweiterung (.psafe3) erst dann gespeichert, wenn Sie den Befehl Speichern oder Speichern unter im Menü Datei ausgeführt haben. Wenn Sie die Anwendung verlassen wollen ohne die Datenbank abgespeichert zu haben, werden Sie sicherheitshalber aufgefordert Sie abzuspeichern.
  • Zusätzlich zu dem hier beschriebenen Erstellen und Wiederherstellen wie es hier beschrieben ist, wird Password Safe automatisch eine Kopie der Datenbank vorhalten, bevor Änderungen abgespeichert werden. Diese Kopie wird im selben Verzeichnis wie die aktuelle Datenbank abgelegt. Sie hat denselben Namen aber eine andere Erweiterung (".ibak" anstelle von ".psafe3"). Siehe hier für mehr Information über automatische Sicherungen.
