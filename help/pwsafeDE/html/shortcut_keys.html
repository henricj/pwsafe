Tastatur Schnelltasten/Beschleunigungstasten

Hier sind die Details und die Einschränkungen für den Einsatz von benutzerdefinierte Tastenkürzel, die Sie mit der Tastatur in Password Safe definieren können, beschrieben.

Die Tastenkombinationen wurden an einer UK-Tastatur mit 102 Tasten getestet. Die Standardzeichen sollten an jeder Tastatur mit Lateinischen Zeichen funktionieren. Einige Tastenkombinationen könnten unterschiedlich zu anderen sein. Z.B. Auf einer UK-Tastatur hat das Zeichen # eine eigene Taste (zusammen mit Tilde ~), aber auf einer US-Tastatur is dies das Zeichen über der 3 (d.h. Umschalt+3). Bitte lesen Sie die Tabelle und wandeln Sie die geeigneten Zeichen für Ihre eigene Tastatur um. Sie können stets ausprobieren ob eine spezifische Tastenkombination für Sie auch funktioniert.

Abhängig von der Sprache, sind bestimmte Alt Tasten reserviert. Diese stimmen mit den Zeichen überein um Punkte im Hauptmenü aufzurufen. Das Drücken der Alt Taste wird diese Zeichen unterstrichen darstellen. Im Englischen Menü sind dies Alt+F (File), Alt+E (Edit), Alt+V (View), Alt+M (Manage) und Alt+H (Help). In anderssprachigen Anwendungen sollten Sie die unterstrichenen Zeichen im Hauptmenü bestimmen, und als reserviert betrachten. Z.B. In der Deutschen Übersetzung wird "File" im Menü in "Datei" umgesetzt, und damit wäre Alt+D reserviert anstelle von Alt+F, usw.

Die Eingabe Taste ist ebenso reserviert, weil sie im Menü Verwalten->Optionen->Schnelltasten als Schnelltaste benutzt wird um den gesetzten Wert zu speichern. Darum kann diese Taste nicht als Schnelltaste für irgendeine Funktion hergenommen werden. Beachten Sie, dass im Modus "Baumstruktur" das Einklappen und Zuklappen der Gruppen mit der Auswahl der Gruppe und der Eingabetaste erreicht werden kann. Dessen ungeachtet, könnte dies zu jede andere gültige Kombination umgesetzt werden, aber die Eingabetaste bleibt weiterhin aktiv, weil es eine Eigenschaft der Prozedur für die Baumstruktur ist. Darum kann auch diese Funktion im Optionsmenü für die Schnelltasten nicht abgeändert werden.

Die Escape Taste ist reserviert, weil sie zum Schliessen des Suchfeldes oder zum Minimieren von Password Safe im Infobereich benutzt wird. Diese Taste ist in jedem Dialog auch equivalent zu "Abbrechen".

Konzeptionell ist die Tastenkombination zum Anschauen eines Eintrages in NUR-Lesen-Modus und zum Bearbeiten eines Eintrages im Schreib-Lese-Modus identisch. Sie können nicht getrennt gesetzt werden. Genau so verhält es sich auch mit den Kombinationen für "Eintrag löschen" bzw. "Gruppe löschen" und "Eintrag umbenennen" bzw. "Gruppe umbenennen". Sie werden darum im Menü "Schnelltasten" nur als "Löschen" und "Umbenennen" dargestellt.

Einige Tasten sind von Windows selber reserviert - z.B. Strg+Alt+Löschen; Tab; Umschalt+Tab; Alt+Tab; Alt+Umschalt+Tab usw.

Nicht-standardmäßige Tasten werden nicht unterstützt (z.B. die Tasten die nur an einer Kanji Tastatur vorkommen).

Password Safe kann keine Unterscheidung zwischen der linke und der rechte Alt, Strg und Umschalt Taste machen. Dies kann dazu führen, dass einige Kombinationen an bestimmte Tastaturen nicht zur Vefügung stehen. Z.B. Das Euro Symbol "€" ist auf einer UK-102-Tastatur die rechte Alt (Alt-Gr) und die Taste 4. Allerdings bringt das Drücken dieser Kombination die Kombination Strg+Alt+4 hervor. Dies liegt an der Art wie Microsoft die Schnelltasten abarbeitet, und ist keine Einschränkung von Password Safe.

Hinweis: Das Setzen von nicht-standardmäßige Kombinationen kann dazu führen dass diese Kombinationen an einem anderen Computer mit einer anderen Tastatur nicht dargestellt werden können. Es wird empfohlen nur Kombinationen aus der unten stehenden Liste zu entnehmen.

Notizen:

  1. Weil die Eingabe Taste für das Setzen der Schnelltasten verwendet wird, kann sie nicht zum Setzen anderer Funktionen eingesetzt werden.

  2. Strg+Q und Alt+F4 sind für das Verlassen der Anwendung Password Safe vorgesehen.

  3. F1 ist für die Hilfe reserviert.

  4. Dies kann abhängig von der Tastatur sein.
