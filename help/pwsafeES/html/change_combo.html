Cambiar las contraseñas maestras

Para cambiar una contraseñas maestra ("Combinación de seguridad"), en primer lugar abre la base de datos cuya contraseña deseas cambiar. Una vez abierta, ve al menú Herramientas y selecciona Cambiar contraseña maestra. Debes introducir la contraseña maestra vigente, y después la nueva contraseña maestra (dos veces, para confirmarla). La contraseña maestra puede introducirse con el teclado físico o el virtual.

Password Safe comprueba si la contraseña maestra es débil. Por lo general, una contraseña maestra (como cualquier contraseña) debe estar compuesta de una mezcla de mayúsculas y minúsculas con al menos un numero o carácter especial (como los que se encuentran por encima de las teclas numéricas en el teclado). Si se introduce una contraseña maestra que el programa considera débil, se le solicitará confirmación. Tienes la opción de aceptar la contraseña introducida (débil) o introducir una nueva.

Notas:
  • Password Safe no tiene un mecanismo para recuperar contraseñas de seguridad perdidas u olvidadas.
     Esta es una decisión de diseño deliberada, ya que cualquier mecanismo para recuperar las contraseñas perdidas plantea un agujero de seguridad que pueden ser explotado..
  • Elige una contraseña difícil de adivinar. Una contraseña trivial, como "contraseña" o tu nombre es fácil de adivinar e invalida la seguridad que proporciona Password Safe.
