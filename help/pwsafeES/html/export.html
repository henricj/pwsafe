Exportar entradas

Password Safe permite exportar (e importar) las contraseñas almacenadas a varios formatos. Para exportar datos utiliza el menú Archivo->Exportar a. Los formatos soportados son:

  • Password Safe v1 (para usar con las versiones 1.x de Password Safe)
  • Password Safe v2 (para usar con las versiones 2.x de Password Safe)
  • Texto sin formato
  • XML

Para versiones previas de Password Safe, selecciona el formato deseado e indica el nombre del archivo de base de datos que quieres crear.

Exportar a texto

Nota: El texto exportado es una copia desprotejida de las entradas seleccionadas. Borrar esta copia tras su uso no es suficiente para hacerlo inaccesible a un atacante determinado, de hecho pasa por alto la seguridad del programa.
Por favor NO utilices esta función a menos que entiendas y aceptes los riesgos.

Password Safe te permite exportar entradas de contraseñas a archivos de texto. Estos archivos pueden ser utilizados y modificados por otras aplicaciones.
Antes de exportar la base de datos a un archivo de texto plano sin encriptar se te pedirá que introduzcas la contraseña maestra o combinación de seguridad.



Puedes indicar algunas opciones:

  • Las entradas de contraseñas se exportan con un tabulador entre campos, esto es, entre los campos grupo.título, usuario, contraseñas y notas. "Grupo.título" se considera un único campo. Todd campo que contenga un espacio en blanco se delimitará con comillas dobles. Sólo el campo notas puede contener entradas con varias líneas. El tratamiento normal consiste en colocar cada línea de notas en una línea independiente en el campo de texto exportado. Sin embargo, esto hace que sea difícil ordenar o comparar dos archivos exportados diferentes. Puedes especificar un delimitador para separar las líneas múltiples al exportalo. Si vas a importar un archivo de texto creado con un delimitador, entonces debe ser especificado, todas las líneas de notas aparecerá como una sola línea en la base de datos resultante.
    Si es necesario, marca que quieres poner por separado todas las líneas en el campo notas en una línea e introduzca el carácter que se utiliza como delimitador. Puesto que todo el campo notas está encerrado entre comillas dobles, no se puede seleccionar este caracter como delimitador de líneas.

  • Tienes más control sobre el formato de exportación haciendo clic en "Opciones avanzadas". Aparecerá el siguiente cuadro de diálogo:

Este cuadro de diálogo te permite exportar sólo algunas de las entradas, filtrar por el texto de un determinado campo, y/o especificar los campos que se van a exportar. Haga clic en Aceptar cuando hayas terminado, o en Cancelar para volver al cuadro de diálogo de exportación de texto sin ningún cambio.

A continuación se te pedirá que proporciones el nombre del nuevo archivo y confirmar sobrescribir un archivo del mismo nombre si ya existe en el directorio especificado.

Nota: Los detalles sobre el formato del archivo de texto exportados se pueden encontrarse en la sección "Formato de exportación e importación de texto sin formato".

XML

XML (eXtensible Markup Language) es un formato popular para transferir datos entre aplicaciones. Password Safe puede exportar (e importar) entradas de la base de dato en este formato. La estructura (o esquema) con la que los datos se exportan y validan se define en el archivo pwsafe.xsd, que se proporciona como parte de la distribución. Este archivo es utilizado por Password Safe al importar datos XML, y debe proporcionarse a las aplicaciones que deseen utilizar los datos exportados.
Nota: El texto exportado es una copia desprotejida de las entradas seleccionadas. Borrar esta copia tras su uso no es suficiente para hacerlo inaccesible a un atacante determinado, de hecho pasa por alto la seguridad del programa.
Por favor NO utilices esta función a menos que entiendas y aceptes los riesgos.
La visualización de los archivos XML se especifica mediante pwsafe.xsl - un archivo de eXtensible Stylesheet Language. Un archivo de ejemplo, que puede ser modificado para sus propias necesidades, se suministra como parte de la distribución. Este archivo es sólo un ejemplo y puede modificarse fácilmente para mostrar los campos que quieras mostrar. Dado que sólo muestra los datos y no los altera, todavía podrás importar a Password Safe posteriormente.

Como con las opciones de exportar a texto plano, puedes controlar que entradas y campos exportar al archivo XML pulsado en el botón Avanzadas del diálogo exportar a XML. Esto mostrará el siguiente cuadro de diálogo:


Este cuadro de diálogo te permite exportar sólo algunas de las entradas, filtrar por el texto de un determinado campo, y/o especificar los campos que se van a exportar. Haga clic en Aceptar cuando hayas terminado, o en Cancelar para volver al cuadro de diálogo de exportación de texto sin ningún cambio. Ten en cuenta que el esquema XML requiere el título y campo de contraseña, por lo tanto, estos no pueden ser deseleccionados.

A continuación se te pedirá que proporciones el nombre del nuevo archivo y confirmar sobrescribir un archivo del mismo nombre si ya existe en el directorio especificado.
