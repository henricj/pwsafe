Importar entradas

Password Safe te permite importar información de contraseñas de otros formatos. Estos archivos pueden haber sido generados por otras aplicaciones, exportados de una hoja de cálculo, o incluso escritos a mano con el bloc de notas o un editor de texto similar.

Los formatos soportados por Password Safe son:

  • Texto
  • XML
  • KeePass (otra utilidad de gestión de contraseñas)

Formato texto

Los archivos de texto utilizados para importar las entradas de contraseñas constan de una o más entradas contraseña. Cada campo dentro de una entrada está separada por un separador de campo y cada entrada puede consistir en una o más líneas. Detalles sobre el formato esperado del archivo de texto a importar puede encontrarse en la sección "Formato de importación y exportación como texto sin formato".

Diálogo opciones de importación de texto

El separador de campo utilizado en el archivo de texto importado y un nombre de grupo predeterminado para importar las nuevas entradas pueden configurarse en antes de procesar el archivo.

  1. Archivo > Importar de....
  2. Seleccionar Texto sin formato...
  3. Fijar los parametros y opciones para importar el archivo de texto
  4. Si se ha utilizado un carácter delimitador de líneas en el campo notas, ya sea en una exportación previa o con un editor de texto, introdúzcalo aquí para que, durante el proceso de importación, el campo de notas se divide en varias líneas mediante la sustitución de este caracter por un nueva línea.
  5. Pulsa Aceptar
  6. Selecciona el archivo de texto a importar.
  7. Al final del proceso de importación, aparece un pequeño cuadro de diálogo mostrando cuántas entradas se han añadido, y cuántas (en caso de haberlas) se analizaron sin éxito.
IMPORTANTE: La primera línea del archivo que se importa debe ser una lista separada por tabulaciones de los nombres de los campos que contiene cada entrada, por ejemplo "Grupo/Título Usuario Contraseña URL", etc. Esta línea se genera automáticamente por la función "Exportar texto", pero debe añadirse manualmente para archivos creados con otras herramientas.

XML

XML (eXtensible Markup Language) es un formato popular de trasferencia de datos entre aplicaciones. Password Safe puede importar archivos XML que se generasen de acuerdo al esquema definido en el archivo pwsafe.xsd, que se proporciona como parte de la distribución.
