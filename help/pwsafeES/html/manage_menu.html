Menú herramientas

El menú herramientas es donde se controlan todas las opciones de Password Safe.

Cambiar contraseña maestra

Seleccionar esta opción abre el dialogo de de cambiar contraseña maestra.
Ver Chambiar las contraseñas maestras para más información.

Hacer copia de seguridad

Crea una copia de seguridad de la base de datos actual.
Ver Haciendo y restaurando copias de seguridad para más información.

Restaurar desde copia de seguridad

Restaura una base de datos de contraseñas desde una copia de seguridad.
Ver Haciendo y restaurando copias de seguridad para más información.

Opciones

Al seleccionar Opciones se abre un dialogo con puestañas donde pueden consultarse y editarse muchas de las opciones de configuración de Password Safe. Para aprender más sobre una pestaña específica selecciónala más abajo:

Pestaña copias de respaldo
Pestaña visualizar
Pestaña seguridad
Pestaña sistema
Pestaña política de contraseñas
Pestaña historial de contraseñas
Pestaña Misc
Pestaña atajos
Nota:
Muchas de las opciones se guardan como preferencias, como se describe aquí.

Validar

Tras seleccionar una base de datos, se comprobará que es completamente compatible con PassowrdSafe.
Nota: Esta opción de menú sólo estará habilitada si no hay abierta ninguna base de datos.

Personalizar barra de herramientas principal

Te permite seleccionar que comandos aparecen en la barra de herramientas y en que orden.

Generar contraseña

Te permite generar una contraseña con la política de contraseñas especificada y ponerla en el portapapeles, pulsando Generar. La contraseñas en el portapapeles puede usarse por alguna otra aplicación.
