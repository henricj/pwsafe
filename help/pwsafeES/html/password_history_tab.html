Administrar el historial de contraseñas



Cuando cambias la contraseña de una entrada,Password Safe puede guardar las contraseñas previas asociadas con la entrada. Esto puede ser útil por ejemplo, si el cambio no se lleva a efecto en el sitio asociado con este entrada. Esta opción indica si las contraseñas recientes se almacenan por defecto en las nuevas entradas. Siempre puedes modificar esto para una entrada dada en el diálogo Añadir entrada o Editar/Ver entrada.

Guargar N contraseñas anteriores por entrada

Seleccionar esta casilla y fijar el valor correspondeinte define la retención de contraseñas por defecto para las nuevas entradas.

Organizar historial de contraseñas de las entradas actuales

Esta sección te permite modificar el historial de contraseñas de las entradas existentes como sigue:
  • Sin cambios - como dice.
  • Dejar de guardar contraseñas previas - Este es el equivalente de desmarcar la casilla "Guardar N contraseñas anteriores" en todas las entradas de la base de datos. No se perderá ningún historial, sólo dejarán de guardarse los siguentes cambios hasta que se reactive guardar el historial.
  • Empezar a guardar contraseñas previas - Es el equivalente de fijar la opción "Guardar Ncontraseñas anteriores" en todas las entradas de la base de datos. Las entradas sin historial guardarán hasta el número anterior de contraseñas. Las entradas con historial de contraseñas seguirán los cambios, pero el número de entradas guardadas no cambiará.
  • Establecer número máximo de contraseñas guardadas al valor de arriba - Todas las entradas, permiten guardar hasta el número anterior de contraseñas. Sin embargo, no se perderá historial si una entrada siten más de este número de entradas almacenadas.
