Pestaña seguridad

La pestaña seguridad da acceso a opciones de seguridad controladas por el usuario. La facilidad de uso y seguridad del programa están afectadas por las opciones listadas aquí, mero no la base de datos encriptada subyacente.

Borrar portapapeles al minimizar

Estando seleccionado, el programa borra el contenido del portapapeles de cualquier dato almacenado (como una contraseñas) al minimizar la aplicación. Esto evita que otra persona o programa lea el contenido del portapapeles. Seleccionando esta opción con la acción de doble click "Copiar contraseña al portapapeles, minimizar" de la pestaña de opciones Misc ocasionará resultados inesperados (la contraseña se copiará, pero inmediatamente se borrará del portapapeles).

Borrar portapapeles al salir

Estando seleccionado, el programa borra el contenido del portapapeles de cualquiere dato almacenadao (como una contraseña) cuando el programa se minimiza o cierra. Esto evita que otra persona o programa lea el contedido del portapapeles.

Confirmar copia de elemento al portapapeles

Estando seleccionado, se pide confirmación cada vez que una contraseña se copia de la base de datos al portapapeles.

Bloquear base de datos al minimizar

Estando seleccionado, bloquea la base de datos de contraseña cuando el programa se minimiza. Cuando se vuelve a maximizar, se requiere la contraseñas para acceder al programa.

Bloquear base de datos al bloquear equipo

Selección esta opción hace que la base de datos de contraseñas se bloquee cuando se bloquea la sesión, previniendo así accesos no autorizados por usuario que pudieran tener acceso al equipo.

Bloquear base de datos después de...

Seleccionar esta opción y fijar un intervalo de tiempo hace que Password Safe bloquee la base de datos de contraseñas si transcurre el intervalo de tiempo fijado sin ninguna acción del usuario en la aplicación.
