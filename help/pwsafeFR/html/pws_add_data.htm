Add Data Window
(Ajouter un enregistrement)

Dans cette fenêtre, vous pouvez ajouter un enregistrement dans la base. Les champs de cette fenêtre sont :

Le champ Group vous permet d'organiser vos enregistrements en groupes logiques. Tous les enregistrements appartenant au même groupe apparaissent sous le nom du groupe, dans la vue Arbre (Tree view) de la fenêtre principale. Notez qu'une hiérarchie de groupes peut être définie, les sous groupes étant séparés par des points. Par exemple, "Finance.Banques" organisera un enregistrement dans le sous-groupe Banques du groupe Finance. Ce champ est optionnel, s'il est vide l'enregistrement sera placé au niveau le plus haut.

Le champ Title est le nom de l'enregistrement sous lequel il sera affiché. Ce champ ne peut pas être vide.

Le champ Username est utilisé pour enregistrer le nom de l'utilisateur associé à cet enregistrement. Si vous définissez un default username dans la boite de dialogue Options, il apparaîtra ici. Ultérieurement, vous pouvez copier le nom d'utilisateur dans le presse-papier.

Le champ Password est celui dans lequel le mot de passe est défini pour un enregistrement. Ce champ ne peut pas être vide. Si vous cliquez sur le bouton Generate, ce champ sera automatiquement rempli avec un mot de passe, généré de façon aléatoire en respectant les règles que vous pouvez définir dans l'Option Password Policy. Ces règles peuvent éventuellement être annulées en utilisant le bouton Override Policy. Si c'est le cas, la boite de dialogue Password Policy sera affichée avant la génération automatique du mot de passe. Les changements effectués alors ne sont pas conservés. Cette option peut être utile, par exemple, si vous avez besoin d'un mot de passe sur un site qui n'obéisse pas aux règles que vous avez défini. Notez qu'en utilisant le bouton Cancel dans la boite de dialogue Password Policy, le mot de passe sera généré en utilisant la règle générique.

Vous pouvez utiliser le champ Notes pour tout autre information dont vous auriez besoin de vous souvenir à propos de cet enregistrement -- telle qu'une adresse URL ou un numéro de téléphone. Si vous entrez une adresse URL commençant par "http://", "https://" ou "ftp://", vous pourrez alors directement parcourir cette URL en effectuant un clic-droit sur l'enregistrement depuis la fenêtre principale.
