Add a Username to All Entries?

Cette boite de dialogue apparaît lorsque vous ouvrez une base de mot passe sans aucun utilisateur dans les enregistrements.

Cliquer sur Yes pour obtenir une boite de dialogue qui vous permettra d'entrer le nom d'utilisateur qui sera ajouté à chaque enregistrement. Cliquez sur No pour laisser la base de mots de passe intacte.
