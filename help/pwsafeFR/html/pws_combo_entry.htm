Safe Combination Entry
(Saisie de la combinaison)

Cette fenêtre apparaît lorsque vous démarrez Password Safe, lorsque vous ouvrez une autre base de données ou que vous restaurez une archive. Entrez la combinaison et appuyez sur la touche Entrée. Les combinaisons sont différentes entre les majuscules et les minuscules ; la combinaison bonjour est différente de BONJOUR.

Si vous entrez trois fois de suite une combinaison incorrecte, le message Safe Combination Entry Error est affiché. Cette fenêtre vous permet d'essayer une autre base ou d'en créer une vide.
