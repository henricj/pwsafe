Safe Combination Entry Error
(Erreur de saisie de la combinaison)

Cette fenêtre apparaît si la combinaison a été saisie incorrectement trois fois de suite. Si vous avez simplement effectué une erreur de frappe, vous pouvez réessayer. Vous pouvez également cherchez une autre base de mots de passe à ouvrir ou en créer une complètement nouvelle et vide.

Si vous avez oublié votre combinaison, quittez la base de mots de passe pour l'instant. Vous pourrez la ré-ouvrir plus tard lorsque vous vous souviendrez du mot de passe. Attention, rien ne peut être fait si vous avez perdu la combinaison. La sécurité du système repose sur le fait que la base de mots de passe ne peut pas être ouverte sans la bonne combinaison.
