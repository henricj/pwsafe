Copy Password to Clipboard Dialog
(Fenêtre Copier le mot de passe dans le presse papier)

Ce message vous informe que le mot de passe sélectionné a été copié dans le presse papier. Vous pouvez à présent directement le copier dans l'application, la fenêtre ou le site Internet qui attend ce mot de passe.

Si vous coller le mot de passe dans une fenêtre, l'option Coller ne sera probablement pas disponible, mais vous pouvez toujours coller en utilisant le raccourci clavier Control-V ou Shift-Inser.
