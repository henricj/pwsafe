Edit/View Entry
(Editer/Voir un enregistrement)

Dans cette fenêtre vous pouvez changer les informations sauvegardées dans un enregistrement. Par défaut, le mot de passe sera remplacé par une ligne d'astérisques. Pour visualiser le mot de passe enregistré, cliquez sur Show password by default in edit mode.

Pour une description des champs de cette boite de dialogue, voir l'aide Add Entry.

Si vous avez fait une erreur et désirez quitter cette fenêtre sans enregistrer les changements, utilisez Cancel. Pour sauvegarder vos changements, cliquez sur le bouton OK.
