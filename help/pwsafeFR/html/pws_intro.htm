Password Safe

Combien de mots de passe et d'utilisateurs devez-vous vous rappeler ? Il y a de fortes chances que la réponse soit "beaucoup." Cela était déjà assez difficile quand vous aviez seulement besoin d'un mot de passe pour accéder à votre compte réseau ou Unix; maintenant beaucoup de sites Internet requièrent un mot de passe également. La plupart des personnes utilisent alors une liste écrite de mots de passe (peu sûr) ou utilisent toujours le même mot de passe sur de nombreux systèmes (très peu sûr). Maintenant, avec Password Safe, vous pouvez garder une liste principale de mots de passe solidement cryptée sur votre ordinateur. Une combinaison simple les ouvre tous.

Password Safe protège vos mots de passe avec l'algorithme Blowfish de Bruce Schneier, une alternative rapide et libre à DES et IDEA. Les détails sur l'algorithme Blowfish, y compris des comparaisons de vitesse et une liste étendue de produits qui emploient Blowfish, sont disponibles sur http://www.schneier.com/blowfish.html

Password Safe vous permet d'avoir plusieurs bases de données de mots de passe. Par exemple, vous pouvez stocker vos mots de passe Internet dans une, vos mots de passe de compte Unix dans une autres, etc... N'oubliez pas de sauvegarder vos diverses bases de données souvent. Si vous souhaitez avoir seulement une base de données, aucun problème. Password Safe prendra soin de tout le détail pour vous et vous n'aurez plus à vous tracasser de la gestion des fichiers.

Commencer

Vous pouvez commencer à protéger vos mots de passe en trois étapes simples.

1. Créer une combinaison pour votre coffre-fort :

Lors de la première utilisation Password Safe vous demandera de créer une combinaison. La combinaison est comme un mot de passe; elle peut inclure des lettres, de la ponctuation ainsi que des chiffres. Votre combinaison protège tout votre coffre-fort de mots de passe, aussi n'utilisez pas comme combinaison quelque chose de facile à deviner -- particulièrement si quelqu'un d'autre pourrait obtenir l'accès à votre ordinateur. Les combinaisons sont sensibles à la différence entre majuscules et minuscules; bonjour est différent de BONJOUR.

Lorsque vous avez choisi votre combinaison, tapez la dans la boite de dialogue intitulée Safe Combination, et ressaisissez la dans la boite nommée Verify. Puis cliquez sur OK. Le boutton OK ne sera actif que si la combinaison a été saisie de façon identique les deux fois.

Lorsque vous démarrez Password Safe, il ouvrira (à condition de saisir la bonne combinaison bien sur !) la dernière base de données utilisée. Si cette dernière ne peut être localisée, vous pourrez en rechercher une autre ou en créer une vide.

2. Saisir les mots de passe :

Après avoir créer votre combinaison, vous pouvez commencer à saisir vos mots de passe. Sélectionner simplement le menu intitulé Add Entry ou cliquez sur le bouton équivalent dans la barre d'outil. Entrez le nom du service associé à ce mot de passe, le nom d'utilisateur, et enfin le mot de passe. Vous pouvez définir un utilisateur par défaut dans le menu Options.

Vous pouvez utiliser le champ Notes pour saisir tout autre information utile au sujet de ce service -- numéros de téléphone, adresse URLs, etc.

3. Récupérer les mots de passe :

Pour copier un mot de passe vers le presse-papier de Windows, double-cliquez simplement sur son titre. Vous pouvez également sélectionner le titre et choisir le menu Copy Password to Clipboard. Vous pouvez alors coller le mot de passe dans toute application ou fenêtre (Si l'application ne fournit pas de commande Coller, vous pouvez cependant toujours coller le mot de passe en pressant Control-V ou Shift-Insert.)

Par défaut, le Presse Papier sera effacé lorsque vous quittez Password Safe.
