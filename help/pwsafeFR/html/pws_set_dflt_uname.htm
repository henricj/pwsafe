Set Default Username? (Définir le mot de passe par défaut)

Cette boite de dialogue vous permet de définir le nom d'utilisateur que vous venez d'entrer comme le nom d'utilisateur par défaut. Le nom de l'utilisateur par défaut n'est pas affiché dans la fenêtre principale mais il est automatiquement ajouté dans la boite de dialogue Add.

Vous pouvez modifier le nom d'utilisateur par défaut actuel dans la boite de dialogue Options.
