备份与恢复数据库

由于存储在密码数据库中的信息通常是敏感和重要的， Password Safe 为备份与恢复密码数据库提供了不同的机制。该章节描述了如何手工生成密码数据库备份。 Password Safe 同样可以配置在每次保存数据库时自动生成数据库备份。这样的备份被命名为中间备份并描述在这里。

生成备份

安全的创建一个当前密码数据库副本。

  1. 单击生成备份
  2. 为备份文件选择一个位置
  3. 单击保存

备注: 强烈建议经常生成密码数据库备份文件，这样将帮助保护密码列表防止系统事故或文件被破坏。 备份加密，使用相同的主密码作为当前数据库备份。

恢复备份

恢复一个备份的密码数据库文件。

  1. 单击恢复备份
  2. 定位备份文件的位置和选择文件进行恢复
  3. 单击打开
    将出现密码提示对话框。
  4. 输入数据库密码并单击确定
备注:
  • 恢复的数据库具有“保存的” Password Safe 标准数据库扩展名 (.psafe3)。在文件菜单选定保存或另存为， 如果退出程序之前没有对文件作出保存操作，程序将提示 "保存" 数据库和为它选定一个名称。
  • 在这里除备份与恢复描述之外，当保存一个更改之前 Password Safe 将自动的保存一个数据库副本， 副本使用除扩展名外其余与当前数据库相同的目录和名称(备份扩展名为 ".ibak" 代替保存的数据库 ".psafe3")。查看这里了解更多关于自动备份的信息。
