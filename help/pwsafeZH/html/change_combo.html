更改主密码

需要更改一个主密码 (“保险箱密码”)，首先打开想要更改密码的数据库。打开后转到管理菜单和选择 更改保险箱密码。输入当前主密码，然后输入新的主密码(两次，为了确认)。

Password Safe 存在一个它认为是薄弱主密码的检查设置。一般来说，一个主密码 (如任何密码) 应该包含混合大小写字符和至少一个数字或特殊字符(如键盘数字键上方的符号)。 如果所输入的主密码被程序认为是弱强度密码，将被提示确认。可以选择接受所输入(弱强度)密码或重新输入新组合密码。

备注:
  • 没有任何途径在 Password Safe 里找回丢失或被遗忘的保险箱密码。
     这是一个故意设计的决定，因为使用任何机制来检索丢失的密码所造成的安全漏洞都可能被利用。
  • 请选择一个让人难以猜测的密码。如果是一个很平凡的密码，例如:“密码”或你的名字是很容易被推测和否定了 Password Safe 提供的安全性。
