历史密码

Password Safe 允许用户根据所设定的日期保存最近一些用于某一项目的密码。 这是有益于显示符合以密码策略作为所需某些工作场所和如果为某些更改在没有奏效的情况下恢复一个早期密码。 访问一个项目的历史密码可通过单击编辑项目->附加选项卡对话框中的“历史密码” 按钮。

该对话框显示最近一些关联特定项目的早期密码。需要显示这些，在编辑/查看项目对话框单击“附加”按钮。用户可以根据下列操作 :

保存 N 个最后密码

如果用户选定该复选框，则该项目最近密码将被保存。也可以规定保存多少密码。
备注: 不选定该复选框将不清除历史密码，将只停止 Password Safe 为该项目存储更多密码。

复制一个密码

单击最近密码列表中的一行将复制密码显示器中的该行到剪贴板。

清除历史密码

单击该按钮将清除这个项目所存储的早期密码。
备注: 在编辑/查看项目对话框点击确定按钮之前不作出实际的修改。 如果错误的单击清除历史密码，在编辑/查看项目对话框中只能单击取消，而不能点击确定按钮。

全部复制

单击该按钮将复制项目历史密码列表(日期和密码)到剪贴板。
