升级早期版本的 Password Safe

为了支持新的功能和增强安全性，在主要版本之间用来存储项目的数据库格式和参数选择已经改变。然而，还有许多已经被努力改进的确保自动和显而易见的升级:
  • 1.x 版本的 Password Safe 只能读取1.x.生成的数据库。
  • 2.x 版本的 Password Safe 可以读取自身建立的数据库及1.x版本 Password Safe 生成的数据库。
  • 3.x 版本的 Password Safe 可以读取所有早期版本建立的数据库。
在所有情况下，读取一个由早期版本 Password Safe 生成的数据库， 只是打开它像所有数据库一样，项目将正常显示，可能 "未知" 字段值不被早期版本支持 (例如: 建立一个项目的日期)。
当保存这样一个数据库时，将被提醒新数据库不能被早期版本的 Password Safe 读取的事实。

生成早期版本的数据库

用户可以建立能通过旧版本 Password Safe 读取的数据库并明确输出为想要的格式(文件 > 输出和选择V1 旧版本格式...或V2 旧版本格式...)。 请注意输出的数据库将不包含附加在新版本中的信息。例如：一个版本 2 的数据库将没有涉及项目修改时间的信息。
